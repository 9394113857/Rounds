DefaultDict:-
A DefaultDict is also a sub-class to dictionary. It is used to provide some default values for the key that does not exist and never raises a KeyError.

Syntax:
class collections.defaultdict(default_factory)

default_factory is a function that provides the default value for the dictionary created.
If this parameter is absent then the KeyError is raised.

Initializing DefaultDict Objects:-
DefaultDict objects can be initialized using DefaultDict() method
by passing the data type as an argument.
